## About The Project

Simple Weather API built with React. I hadn't built a Weather API since bootcamp, so I decided to try to make one on my own with React. I also wanted an opportunity to try a new idea for I had for interface styling, which I used with this app. The app includes CSS Transition animations that make page transitions smooth.

<!-- GETTING STARTED -->

## Getting Started

To use the app, simple visit the [Vercel Deployment](https://weather-api-ashy.vercel.app/) and search for a location. The app will then retrieve weather information for the provided location including current weather, an hourly forecast, and a daily forecast.
